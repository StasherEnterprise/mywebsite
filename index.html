    • 1. phpMyAdmin: Nieuwe MySQL-gebruiker Aanmaken

    Stappenplan:

    1. Log in op phpMyAdmin:
      • Ga naar http://localhost/phpmyadmin.
      • Log in met je MySQL-beheerdersaccount (bijvoorbeeld de root-gebruiker).
    2. Maak de gebruiker aan:
      • Klik in het menu op Gebruikersaccounts (Users).
      • Klik op Voeg gebruiker toe (Add user account).
    3. Voer gebruikersgegevens in:
      • Gebruikersnaam: Kandidaat
      • Host: Selecteer localhost.
      • Wachtwoord: 3xamen2023!
    4. Database rechten toewijzen:
      • Kies bij Database voor gebruiker de optie:
        • Maak database met dezelfde naam en geef alle rechten
        • OF selecteer de database purple_hrm en klik op Controleer alles.
    5. Globale privileges:
      • Als extra zekerheid kun je bij Globale privileges alleen benodigde rechten aanvinken.
    6. Bevestig:
      • Klik onderaan op Uitvoeren om de nieuwe gebruiker aan te maken.
    7. Controleer toegang:
      • Log uit van phpMyAdmin.
      • Log in met de nieuwe gebruikersnaam Kandidaat en wachtwoord 3xamen2023!.
      • Controleer of de gebruiker toegang heeft tot de database purple_hrm.

    MySQL Query's

    Query 1: Totaal aantal medewerkers in dienst

    SELECT COUNT(*) AS totaal_in_dienst FROM medewerkers (Op het examen kan deze tabel ook anders heten, bv. employees. Maak een lege query en run "SHOW TABLES;" om alle namen van de tabellen te kunnen zien) WHERE in_service = 1; (De "in_service" is de tabel die aangeeft of een medewerker in dienst is, dit kan op het examen ook anders heten bv. "Dienst_status")

    Query 2: Overzicht van medewerkers in dienst

    SELECT firstname, lastname, adres, city, zipcode, email, CONCAT(firstname, lastname) AS username, 'W@chtwoord123!' AS password, d.name AS department FROM medewerkers m JOIN departments d ON m.department_id = d.id WHERE m.in_service = 1;

    Query 3: Medewerkers uit dienst in 2022

    SELECT firstname, lastname, CONCAT(firstname, lastname) AS username, outservice_date FROM medewerkers WHERE outservice_date BETWEEN '2022-01-01' AND '2022-12-31' ORDER BY outservice_date ASC;
